# Focus Aurora 🔥

A colourful pomodoro timer built to help you build daily focus habits — 
with breaks, streaks, lofi music, and themes, all in a single page.

🔗 **Live site:** https://yourusername.github.io/focus-aurora/

## Features

- ⏱ Three timer modes — Concentration, Short break, Long break — each adjustable in 5-minute steps
- 🔥 Daily streak tracker to keep you coming back
- 🎧 10 generated lofi tracks, no downloads or accounts needed
- ⧉ Pop-out timer window so you can keep working in other tabs
- 🌌 Aurora, Midnight, and Sunset themes, plus custom photo backgrounds
- 🌐 Available in English, Spanish, French, Hindi, and Japanese
- 🔔 Auto-start toggles for breaks and focus sessions, with alarm sound options

## Usage

Just open the live site above — no installation needed. All settings and your 
streak are saved automatically in your browser.

## Tech

Plain HTML, CSS, and JavaScript — no frameworks, no build step.
